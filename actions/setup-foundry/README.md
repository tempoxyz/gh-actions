# Setup Foundry

Install Foundry toolchain (forge, cast, anvil, chisel).

## Inputs

| Name | Description | Required | Default |
|------|-------------|----------|---------|
| `version` | Foundry toolchain version | No | `""` |

## Usage

```yaml
steps:
  - uses: tempoxyz/gh-actions/actions/setup-foundry@main
```
